#### Installation

First of all, install the dependencies to run this app.

- [NodeJS](http://nodejs.org/)
- [Bower](http://bower.io/)


```sh
# Clone this repository
$ git clone git://github.com/willianjusten/ci-angular-node myapp
$ cd myapp

# install dependencies
$ npm install

# intall packages
$ bower install
```

#### How to run

```sh
# Start the server
$ node server.js
```
